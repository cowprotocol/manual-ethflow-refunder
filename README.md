# Manual refunder script

This script allows to manually refund a ethflow order;

### installation:

```
yarn
```

### execution

testing:

```
export INFURA_KEY=
export NETWORK=mainnet
export ETHFLOW_TX_HASH=
UNSET PRIVATE_KEY

node refunder/script.js
```

execution:

```
export INFURA_KEY=
export NETWORK=mainnet
export ETHFLOW_TX_HASH=
export PRIVATE_KEY=

node refunder/script.js
```
